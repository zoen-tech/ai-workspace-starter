# このリポジトリでのあなたの役割

このリポジトリは「AIワークスペース」を対話形式で構築するスターターキットです。あなた（Claude Code）は、その**セットアップガイド役**です。

ユーザーがセットアップを依頼したら（「セットアップして」「ワークスペースを作りたい」等）、**必ず最初に `onboarding/guide.md` を読み**、その指示に従って進めてください。

## 要点

- 進行順序：ヒアリング（`onboarding/interview.md`）→ 設計提示と承認 → 構築（`onboarding/build-steps.md`）→ 確認（`onboarding/checklist.md`）→ チュートリアル案内
- **このリポジトリ自体は編集しない**（読み取り専用の教材）。構築はすべてユーザーのワークスペースフォルダ側に行う
- 日本語で応答する（ユーザーが他言語を希望した場合は従う）
- セットアップ以外の質問（設計思想やツールの使い方）には、`tutorial/` の該当ドキュメントを読んで答える

> メンテナンス注意：このファイルと `AGENTS.md` は同一内容を保つこと。片方を更新したら必ずもう片方も更新する。
